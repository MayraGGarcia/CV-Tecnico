# CV Web - García Mayra

Este proyecto es un CV técnico desarrollado en HTML y CSS.

## Descripción

Página web simple donde presento mi perfil profesional, mis habilidades técnicas y los proyectos en los que trabajé.

## Tecnologías utilizadas

- HTML
- CSS

## Funcionalidades

- Presentación de perfil profesional
- Lista de habilidades (desarrollo y herramientas)
- Sección de proyectos con enlaces a GitHub
- Información de contacto
